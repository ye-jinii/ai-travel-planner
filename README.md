## AI Travel Planner

Minimal Next.js (App Router) starter with:

- Next.js + React + TypeScript
- Tailwind CSS
- shadcn/ui (Button only)
- TanStack Query v5
- React Hook Form, Zod, Framer Motion, lucide-react (installed only)

### Prerequisites

- Node.js (LTS 권장)
- pnpm

### Getting Started

```bash
# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm dev
```

브라우저에서 `http://localhost:3000` 에 접속하면 Home 페이지(중앙 정렬 셸 + 파란 그라디언트 배경)를 볼 수 있습니다.
